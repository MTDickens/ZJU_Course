笔者是王晓梅老师班的，王老师真的是妈妈级别的人物，人真的很好，而且给分很好！强烈推荐！笔者因为线上直播未观看，本来很担心点名分数都被扣光，后来询问老师，老师说我们会让大家过一个好年的，呜呜。
